Equipement (
    #numero de serie : integer,
    nature : string,
    type : string,
    modèle :string,
    marque : string,
    prix : integer,
    ) avec tout les attributs NOT NULL
On va supposer que le prix d'un équipement est un nombre entier, et que le numéro de série est la clé primaire car il est unique pour un équipement ce qui va permettre de différencier tout les équipements.

Le_ticket (
    #numero : integer,
    dtPanne :date,
    description : string,
    ) avec tout les attributs NOT NULL
Ici le numéro du ticket est l'élément qui va permettre de différencier tout les tickets, il est donc unique et c'est donc la clé primaire.

L_intervention (
    #numero : integer,
    prix : integer,
    ) avec numero NOT NULL
Tout comme pour Le ticket, le numéro de l'intervention va permettre de différencier les différentes interventions , il est unique et c'est donc la clé primaire.
